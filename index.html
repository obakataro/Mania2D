主な機能

レイヤー管理

ドラッグ&ドロップで画像を追加。親子関係とアンカーをGUIで設定できます。

パラメータを自作

スライダーや2Dパラメータを自由に作成して、レイヤーに連動させます。

キーフレームタイムライン

直感的なタイムラインで、イージングつきの補間を作ります。

出力

制作したモーションは mp4 / webm / gif / png シーケンス / JSON（モデル + モーション）で書き出せます。FFmpeg WASM を使うことでブラウザ内で変換可能です。
